### 그래서 뭘 얻었어? :bulb:
#### 자바의 == vs equals

- == 
: 같은 값을 **가리키는** 지, 값 자체는 같아도 가리키는 **곳(위치)**이 다르다면 false 
- equals 
: 값 **내용**만 같다면 true

> 원시 데이터 값이라면 동등 비교 연산자(==)를, 아니라면 equals를 쓰자
